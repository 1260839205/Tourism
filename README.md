## 黑马旅游网

#### 一、 案例简介

​				本项目学习自bilibili视频网传智播客黑马旅游网项目

#### 二、 需求分析

1. 注册功能
2. 登陆功能
3. 退出功能
4. 导航栏分类展示功能
5. 分页查询功能
6. 数据统计功能
